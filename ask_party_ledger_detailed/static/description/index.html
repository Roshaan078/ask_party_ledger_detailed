Party Ledger Detailed Report

Advanced Accounting Ledger for Odoo

ASKSOL Party Ledger Detailed is a powerful and accurate accounting report designed for Odoo. It provides a complete partner-wise ledger with opening balance, invoices, credit notes, payments, and running balance — all calculated using optimized SQL for high performance.

Key Features

Opening Balance

Correct opening balance calculation based on selected start date, always shown at the top of the ledger.

Accurate Running Balance

Real-time running balance calculated using debit and credit movements.

Invoices & Credit Notes

Supports customer invoices, vendor bills, refunds, and credit notes without forcing sign changes.

Payments & Journal Entries

Includes payments and journal entries linked to receivable and payable accounts.

Date Range Filter

Filter ledger by start and end date with proper opening balance logic.

Professional PDF Output

Clean, readable PDF report with formatted dates and aligned columns.
